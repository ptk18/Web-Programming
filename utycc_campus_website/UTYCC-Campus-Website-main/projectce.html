Third Year

Automatic Whiteboard Cleaner

Introduction

An automatic white board cleaner is a machine which is used to clean board mechanically. This mini project is an automatic whiteboard cleaner for classes (e.g.: school, university). This project is to develop a motorized whiteboard cleaner controlled by an Arduino UNO and 4-DC motors.

Keywords

Whiteboard, Arduino UNO

Aims and Objective

  • To save time and energy
  • To make a smart classroom
  • To provide easy and fast cleaning
Ezzy Robot Arm with Bluetooth Control Carrier

Introduction

The project is a robot arm that can be controlled by android application. Recently, this robot arm is used to carry things to and form. The main devices are Arduino UNO, Bluetooth module (HC-06), motor and motor driver. We control 4 servos that are connected in the 3D printed frame to control the arm through Bluetooth. The user is able to control the car through the mobile car via Bluetooth module.

Keywords

Arduino, Bluetooth, Robot arm

Aims and Objective

  • To carry things for industrial
  • To apply in the hazardous environments
  • Make fewer mistakes than man, saving time

Sumobot

Nowadays, the world wd live has so much garbage. That garbage will cause great harm to the environment and loss of natural resource. Eco-Bin

Introduction

This project solves garbage problem by automatic trash classification of recyclable or not and monitors the bin (now much percent filled inside it) over the internet via IOT. Our solution idea will be the best for solving indiscriminate dumping of garbage. The Eco-Bin recognize the waste via the system positioned inside the bin which uses camera, sensors, image recognition and artificial intelligence (AI).

Keywords

IOT, AI, garbage

Aims and Objective

  • To solve currently confront garbage problems in our country
  • To reduce the pollution caused by recycles waste
  • To be an efficient solution to the current waste management problem
  • To help the recycling process easily and quickly using AI and IOT
