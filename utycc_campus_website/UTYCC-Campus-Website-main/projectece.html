Four Ways controlled Car

Introduction

The project is a car that can be controlled by human and also automatic using four smart ways. They are remote control, voice control, self-control and line follower. The main device is Arduino UNO as the brain of this car. This determines various decisions for controlling the car. Bluetooth module (HC-06) is used for data communication in voice control. Ultrasonic sensor and IR sensor determine the distance for self-control. The signals from IR sensor determine to follow the line.

Keywords

Arduino UNO, Ultrasonic sensor, Bluetooth module (HC-06), IR sensors

Aims and Objective

  •  To design a car that can be controlled by four methods. To become a product as the next step. To understand the use of Arduino, various sensors and components.
Line Follower and Remote Controlled Car

Introduction

Today, electronic devices are taking place in everywhere. Line follower and remote control car, an essential technological supply, is introduced. LM 324 helps the DC motor to control the speed in triangle waves. IR sensors are used to remind the robot when if faces the things from accident. LDR is in another way to help the car. Automatic Gun Aiming System

Introduction

Sensor Based Automatic Aiming System is an automatic gun target and aiming system if found an object within a range of sensors. The project is primarily based on PIR sensors, microcontroller and wireless transmitter and receiving units using FSK. The sensors will sense any living object inside the range, provide the signal to microcontroller, in response, it generates the code on the site and send to the watch tower where the receiver receives the codes, provides that codes to microcontroller, interprets the location of the object corresponding to received code, activates targeting system, buzzer system and at last firing system.

Keywords

Auto aiming system, wireless transmission, wireless receiving, buzzer system.

Aims and Objective

  • To detect any IR radiation in the range of border
  • To target the object through DC motor
  • To use in home security at night by simply adjusting the range
