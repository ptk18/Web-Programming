Second Year

Color Sorting Machine

Introduction

Color sorting machines (called color sorters) separate items according to their colors. We use belt-type color sorting machine because it makes the transmission smooth and stable, it breaks small percentage of materials.

Keywords

12V DC motor, Arduino UNO, Servo motor, color sensor, Conveyor frame

Aims and Objective

This project aim is to separate the color precisely by reducing the human resource, improve product quality and add social benefits. And then, it can save a lot of time. If we use the workers in sorting area, they were being bored and disappointed on their work. Instead of using workers, we can place the Color Sorting Machine.

Introduction

The controllable Cleaning Machine is very much useful in cleaning floors in our daily life. It cleans the floor with liquid soap: size is portable as we can transfer from one place to another easily.

Keywords

Power supply (12V Li-Po Battery), Motors, Motor driver, Arduino UNO, Bluetooth module, Servo motor, Ultrasonic sensor.

Aims and Objective

  • To beautify the floor
  • To save man power
  • Low maintenance and easy to operate.
Rain Sensing Automatic Clothes Hanger

Third Year

Automatic Box Transporting Mechanism

Introduction

This project aims for the utilization of kinematic synthesis to fabricate a working physical model of link transporting mechanism. The box transporting mechanism, operated with crank and links arrangement.

Keywords

Box moving mechanism, DC motor, slider crank mechanism

Aims and Objective

The aim of this project is to fabricate the box transporting mechanism, which can make easier to move boxes form on section to another while processing in the factories, In the workstation, an assembly line in order to obtain the required production rate and to achieve a minimum amount of idle time. The objective is to understand project planning and execution and understand the usage of various mechanical machines tools.

CNC Cake Decoration Machine

Introduction

What is cake decoration process?
Cake decorating is one of the sugar arts that use icing or frosting and other edible decorative elements to make plain cakes more visually interesting. Among them, cake decorating machine may be the one. For example, this machine can be used in café and coffee shop to make cakes attractive. It can draw mostly everything we want. When it is difficult to draw by people, this machine can draw the design that give from the program.

Maze Robot

Introduction

One of the simplest ways to solving a maze with a robot is by using the Wall Follower algorithm, also known as the left hand rule (or right hand rule). First, ultrasonic sensors sense the wall and inform the Arduino and the Arduino drives the dc motor.

Keywords

Arduino, sensors, dc motors

Aims and Objective

The purpose of maze robot is finding its way through any type of maze. Objectives are to understand and implement the wall follower and flood fill algorithm, ultrasonic sensor and Arduino based hardware.

Vacuum gripper can be used in many materials handling application. The main objective is to highlights the importance of vacuum cup universal gripper in industrial robot application. They must be designed more likely as human hand, both in their sensory and control capabilities.
